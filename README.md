# 🛡️ cyart-vapt-team

## 📌 Project Overview

This repository contains all practical tasks and projects related to **Vulnerability Assessment and Penetration Testing (VAPT)**. The work is performed using Kali Linux tools and follows industry methodologies such as PTES (Penetration Testing Execution Standard).

The repository is organized weekly, with each folder containing documentation, workflow steps, screenshots, and reports.

---

## 📂 Repository Structure

* **Week-01** → Introduction to VAPT and environment setup
* **Week-02** → Vulnerability Scanning, Reconnaissance, Exploitation, DVWA SQL Injection using sqlmap
* **Week03** → Full VAPT Cycle (Capstone Project - Exploitation + Reporting)
* **Week04** → Advanced Exploitation, API Security, Privilege Escalation, Network Attacks, Mobile Testing, and Full VAPT Engagement

## 🔁 VAPT Workflow

The following methodology was followed across all tasks:

1. **Reconnaissance**

   * Identifying target systems and services

2. **Scanning**

   * Detecting vulnerabilities using tools like Nikto and OpenVAS

3. **Exploitation**

   * Using Metasploit and sqlmap to exploit vulnerabilities

4. **Post-Exploitation**

   * Gaining system access and verifying privileges

5. **Reporting**

   * Documenting findings and recommending remediation
